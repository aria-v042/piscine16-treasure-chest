`ifconfig` // displays the status  of  the  currently  active interfaces

- `ifconfig -a` // display all interfaces which are currently  available,  even  if down

`grep` // print lines that match patterns

- `grep -o <pattern>` // prints only the part of the line matching the given pattern

`cut` // print selected parts from each line of input

- `cut -d <delimiter>` // select the fields delimiter

- `cut -f <field numbers>` // select only this/these field(s)

> `ifconfig -a | grep -o 'ether [a-fA-F0-9:]*' | cut -d " " -f 2` // print only the second field of each line that begins with "ether" from among the displayed interfaces, which corresponds to the MAC addresses of those interfaces
