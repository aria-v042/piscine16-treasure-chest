Write a command line that displays the list of groups the user (defined in the
environment variable FT_USER) belongs to.

The output should be comma-separated, without spaces.

Examples:
- for FT_USER=bocal:
```
$>./print_groups.sh
bocal,adm,cdrom,sudo,dip,plugdev,lxd,lpadmin,libvirt$>
```

 - for FT_USER=daemon:
```
$>./print_groups.sh
daemon,bin$>
```

----
### 1. Command that displays the list of groups a user belongs to

`id` // print user and group information for each specified USER, or (when USER omitted) for the current user

- `-G, --groups` // print all group IDs

- `-n, --name` // print a name instead of a number, for `-ugG`

> `id -Gn` // displays the list of names of the groups the current user belongs to

### 2. Format the output to meet the ex05 requirements

output should be comma-separated, without spaces AND should not print new line (\n) when the execution is finished

- `tr` // translate, squeeze, and/or delete characters from standard input, writing to standard output

	- `-d` // delete characters in SET1, do not translate

> `id -Gn $FT_USER | tr " " "," | tr -d "\n"` //
