### 1. Generate SSH key

> `ssh-keygen` // generates a SSH key and saves it in `~/.ssh/`

### 2. Copy public key to the `ex03/` dir

pwd: `~/Documents/piscinecore/ex03/`

> `cp ~/.ssh/id_rsa.pub id_rsa_pub` // copies generated public key to `ex03/` with the name `id_rsa_pub`

### 3. Update SSH key on the 42 intranet

> `cat id_rsa_pub` -> copy the content of the file (i.e. your public ssh key)

> Go to https://profile.intra.42.fr/gitlab_users and paste your public SSH key into the form

> [!check] done
