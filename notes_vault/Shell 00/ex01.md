### 1. Create file

 `touch testShell00` -> create file "testShell00" 
 

### 2. Check output of `ls -l`
ref: https://www.computernetworkingnotes.com/linux-tutorials/the-output-of-the-ls-l-command-explained.html

> `ls -l`

out:
```
-rw-r--r-- 1 frodrig2 2026_porto 0 Jan 26 13:50 testShell00
```

>> desired output: `-r--r-xr-x 1 XX XX 40 Jun 1 23:42 testShell00`

to change:
- permissions
- file size
- date/time

### 3. Modify file size
ref: https://superuser.com/questions/609020/how-can-i-create-a-file-with-a-specific-size-from-a-command-line

> `truncate -s 40 testShell00`-> modifies file size (in bytes)

### 4. Modify permissions

> `chmod 455 testShell00` -> modifies permissions to `r--r-xr-x`

### 5. Modify date & time
ref: https://askubuntu.com/questions/62492/how-can-i-change-the-date-modified-created-of-a-fi
