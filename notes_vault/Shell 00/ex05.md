### 1. Command to display IDs of last five commits

> `git log` // displays the commit history of a Git repository, including IDs

#### 1.1. Display only last five commits

> `man git log` // view manual for git log in order to find the relevant options

- `git log -n 5` // displays only the last 5 commits

#### 1.2. Display only the commit IDs

- `--pretty="%H"` // option `--pretty` to format the commit logs with the placeholder `%H` to show only the commit hashes (i.e. IDs)

### 2. Create git_commit.sh executable file

> `vim git_commit.sh` < git log -n 5 --pretty="%H" // create and edit the file that displays the IDs of last five commits
